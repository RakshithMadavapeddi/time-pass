# Hotel Check-In Flow (Vanilla HTML/CSS/JS)

## Run
Open `/Users/rakshith/Desktop/Testing/index.html` in a browser.

## Notes
- The scanner screen uses a simulated AAMVA paste area and success/failure actions.
- Returning guest routing is based on scanned identification number matching a local set.
- Card payment supports a force outcome toggle (Auto / Force Success / Force Decline) for state testing.
- Snackbars are implemented for:
  - Auto-fill failed
  - Required field errors
  - Details auto-filled
